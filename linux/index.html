# 软件源

# git

# 配置 git

git config --global user.name "yourname"
git config --global user.email "youemail@email.com"
git config --global core.editor emacs

# 配置 ssh key

生成 ssh key

ssh-keygen -t rsa -C "youremail@email.com"
cat ~/.ssh/id_rsa.pub

打开 github 网站，打开 setting，选择左侧的 SSH and GPG keys，选择 New SSH key，将生成的 ssh key 粘贴进去保存。 打开终端测试是否成功

ssh -T git@github.com

显示如下信息表明设置成功

# 更改终端字体

下载 Hack Nerd font(powerlevel10k 主题需要) 安装字体 终端右键选择配置文件首选项 选择自定义字体 选择 Hack Nerd font

# 安装powerlevel10kopen in new window主题

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ~/powerlevel10k
echo 'source ~/powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc

# Anaconda3

前往官网open in new window下载 Anaconda

bash ~/下载/Anaconda3-2023.03-1-Linux-x86_64.sh

注意自己的安装路径，我的是 ~/anaconda3 安装完成后执行 bash ~/.bashrc 或者 bash ~/.zshrc 用 conda list 检查是否安装成功 取消自动激活

conda config --set auto_activate_base false

# pip 加速

创建文件 ~/.pip/pip.conf，加入下面的代码

[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple

使用 pip config list 查看配置
